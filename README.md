## 🧯 Troubleshooting
If the app does not start:

- Make sure you downloaded the right file for Windows
- Extract the ZIP file before opening it
- Right-click the app and choose Run as administrator

- Restart your PC and try again

If WeChat does not connect:

- Check your internet connection
- Make sure WeChat is open and signed in
- Close the app and open it again
- Confirm that your QR code or login step finished

If Claude Code does not reply:

- Check your Claude Code access
- Make sure the app still shows a live connection

- Restart the app and test again
